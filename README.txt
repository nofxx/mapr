= mapr

 Create offline maps from online sources.


== DESCRIPTION:

 mapr will create the files you need to transfer to your gadget in orderto browse those maps offline.


== FEATURES/PROBLEMS:

 Gadgets supported:

  * iPhone
  * PSP
  * Any device that supports Map This!"

 Sources supported (planned):

  * Openstreetmaps
  * Live Maps
  * Google Maps
  * Yahoo Maps


== SYNOPSIS:

  mapr <url from provider>
  

== REQUIREMENTS:

* hpricot

== INSTALL:

 gem install nofxx-mapr -s http://gems.github.com
